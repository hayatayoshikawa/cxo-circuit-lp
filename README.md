# CXO CIRCUIT — LP デザイン案アーカイブ

経営人材特化のキャリアプラットフォーム「CXO CIRCUIT（仮称）」の LP デザイン検討アーカイブ。
運営：Beyondge株式会社 ／ 公開目標：2026年7月14日週。

サービス名は CXO CIRCUIT / CXO DRAFT / CXO AGENT が確定候補（最終未決）。

---

## ディレクトリ構成

```
concepts/   コンセプト案（自己完結HTML・Google Fontsのみ／ブラウザで直接開ける）
warm_mock/  現行の実装版（圧縮アセット同梱・そのままホスティング可）
gas/        登録フォーム用 Google Apps Script
```

## デザイン案一覧

### concepts/01_sticker_geometric.html ★本命候補
「かっこいいIT企業×ステッカーコラージュ×動的UI」路線。
- 背景：canvasのネットワーク／パーティクルがマウス追従で常時アニメ
- 特大英字ヘッドライン「BEYOND YOUR TITLE.」（ホログラム箔＋蛍光ライム）
- ステッカー類（バーコードタグ／ホログラム箔ラベル／ネオン付箋／地球シール／×カード）は全てCSS/SVGで自作
- 野上さんFB「黒×金の高級・お金訴求を避け、新しい挑戦をするトンマナに」を反映した方向
- 背景は将来的に動画素材へ差し替え可能

### concepts/02_editorial_kinetic.html
「静かな高級編集 × 強いモーション」路線（INK&PIXEL系）。
- 白基調モノクロ＋極小の朱アクセント、明朝のキネティックタイポ（1文字ずつ落下）
- ライブ東京時計・カスタムカーソル・スクロールリビール・無限マーキー

### concepts/03_dark_circuit.html
「夜のボードルーム × 回路モチーフ」路線。
- ミッドナイト地×真鍮ライン。回路が「現在→経営参画→CXO」を描く
- ※「ハイキャリ・お金重視・他社と被る」との指摘あり（不採用寄り）

### concepts/04_photo_destination.html ★別案（PLANIT/NEXORA系）
「B&W建築写真 × 特大英字 × スーツケースの“○○行き”タグ」路線。
- 背景：金融街の摩天楼をグレースケールで全面配置（Unsplashフリー素材・ゆっくりズーム）
- ヘッドライン「LEAD THE NEXT.」
- 旅×キャリアのステッカー：行き先タグ「TO: 経営」／搭乗券「現在地→経営」／緑バッジ「NEXT STOP 経営行き」／入国スタンプ風「経営行」／ホログラム箔
- 写真は将来 実写差し替え可能（filterでB&W化しているのでカラー写真でも統一感が出る）

### concepts/wireframe.html
③ LP構成要素（全8セクション）のワイヤーフレーム。左＝図／右＝セクション説明の2カラム。

### warm_mock/index.html
初期の和エディトリアル版（アイボリー＋レッド＋ゴールド）。一時 Cloudflare Pages で公開した版。
`cxo_agent_lp_enterprise.html` は企業向けLP。`cxo_agent_assets/` に圧縮済み写真・ロゴ。

## LP構成（全8セクション）

0 ヘッダー ／ 1 ヒーロー（語りかけ型）／ 2 問いかけ ／ 3 できること ／
4 求人イメージ ／ 5 登録後の流れ ／ 6 FAQ ／ 7 最終CTA＋登録フォーム ／ 8 運営会社（最末置き）

確定コピー：「経営に挑むキャリアは、ここから始まる。」

## 登録フォーム

`gas/form.gs` を Google Apps Script にデプロイし、発行URLを各HTMLの `GAS_ENDPOINT` に設定すると、
スプレッドシート記録＋通知メールが動作する。現状は未設定（画面確認のみ）。
